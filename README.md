# Heshan Pravintha — Personal Portfolio v2

> Senior Software Engineer | Mobile Developer | Flutter & React Native

A modern, responsive personal portfolio built with React and Vite — showcasing professional experience, mobile projects, open-source Flutter contributions, and technical skills.

---

## Tech Stack

| Category | Technologies |
|---|---|
| Framework | React 19, Vite 6 |
| Styling | Tailwind CSS, Framer Motion |
| Icons | React Icons |
| Fonts | JetBrains Mono, Poppins |
| Deployment | Netlify / Vercel |

---

## Features

- Responsive design — mobile, tablet, and desktop
- Animated hero section with typewriter effect
- Dedicated Flutter Community Contributions section
- Timeline-based experience section
- Skills categorized by languages, frameworks, and tools
- Smooth scroll navigation with active states
- Production build via Vite with gzip compression

---

## Sections

- **Hero** — Intro with animated typewriter and profile photo
- **About** — Bio and education history
- **Skills** — Languages, frameworks, tools, and core competencies
- **Projects** — Featured production mobile apps (Airtel, Hutch, AIA)
- **Flutter Contributions** — Published pub.dev packages
- **Experience** — Career timeline at Omobio & Fiverr
- **Contact** — Direct email CTA

## Getting Started

```bash
# Clone the repo
git clone https://github.com/HeshanPravintha/my-portfolio-v2.git

# Navigate into the project
cd my-portfolio-v2

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
